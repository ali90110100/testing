What is Gensyn?

Gensyn is a decentralized compute protocol that enables machine learning model training at scale by connecting underutilized compute resources across the globe. It uses cryptographic verification and a novel proof-of-learning system to ensure trustless execution of ML workloads.

Trustless Verification

Uses probabilistic proof-of-learning and the Verde protocol to verify ML work without re-execution, ensuring correctness on untrusted hardware.

Global Compute Network

Connects underutilized GPUs worldwide to create a massive, cost-effective compute resource for AI training at 60-80% lower costs than traditional cloud providers.

Protocol Architecture

Built on a custom Ethereum rollup with four key roles: Submitters, Solvers, Verifiers, and Whistleblowers to coordinate and verify computations.

Reproducible Operations

RepOps ensures bitwise reproducibility of ML operations across different hardware, enabling reliable verification of model training.
